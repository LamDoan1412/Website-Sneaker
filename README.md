👟 Sneaker Website - Trang web bán giày Sneaker
Một ứng dụng web hiện đại, giao diện bắt mắt dành cho việc kinh doanh và quản lý cửa hàng giày Sneaker. Dự án được tối ưu hóa trải nghiệm người dùng từ khôi duyệt sản phẩm, giỏ hàng cho đến thanh toán.

🚀 Tính năng nổi bật
🛒 Dành cho Khách hàng (Frontend)
Trang chủ sinh động: Banner quảng cáo, sản phẩm mới về, sản phẩm bán chạy (Best Seller).

Bộ lọc sản phẩm thông minh: Lọc theo thương hiệu (Nike, Adidas, Jordan,...), giá cả, kích thước (size), và màu sắc.

Chi tiết sản phẩm: Xem hình ảnh chi tiết, chọn size, số lượng và kiểm tra tình trạng kho hàng.

Giỏ hàng & Thanh toán: Thêm/sửa/xóa sản phẩm trong giỏ hàng, áp dụng mã giảm giá (coupon) và tích hợp cổng thanh toán.

Quản lý tài khoản: Đăng ký, đăng nhập, theo dõi lịch sử đơn hàng và trạng thái vận chuyển.

📊 Dành cho Quản trị viên (Admin Dashboard)
Thống kê doanh thu: Biểu đồ trực quan về doanh số theo ngày/tháng/năm.

Quản lý sản phẩm: Thêm, sửa, xóa (CRUD) thông tin giày, số lượng size trong kho.

Quản lý đơn hàng: Cập nhật trạng thái đơn hàng (Chờ duyệt, Đang giao, Đã giao, Hủy).

Quản lý người dùng: Phân quyền (Admin/User) và quản lý tài khoản khách hàng.

🛠️ Công nghệ sử dụng (Tech Stack)
(Bạn hãy giữ lại những công nghệ bạn thực sự dùng và xóa bớt cái thừa nhé)

Frontend: ReactJS / Next.js / VueJS hoặc HTML5, CSS3, JavaScript (Bootstrap / Tailwind CSS).

Backend: Node.js (Express) / PHP (Laravel) / Python (Django / Flask) / Java (Spring Boot).

Database: MongoDB / MySQL / PostgreSQL.

Authentication: JWT (JSON Web Tokens) / Firebase Auth.

Payment Gateway: Stripe / PayPal / MoMo / VNPAY.
